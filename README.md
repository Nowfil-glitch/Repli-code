# Repli-code

Build, learn, and replicate small real-world frontend projects.

This is a minimal static site used for learning and demonstrations.

## What’s included

- `index.html` — markup
- `style.css` — styles
- `script.js` — tiny UI script (smooth scrolling)

## Quick start

```bash
# serve locally with any static server, e.g.:
# Python 3
python -m http.server 5173

# or with Node (http-server)
npx http-server -c-1 .
```
